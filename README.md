# 🍳 RecipeFinder

Welcome to RecipeFinder! This is a simple, beautiful, and easy-to-use web application that helps you discover your next favorite meal. Whether you're looking for a specific dish, want to explore cuisines by country, or just prefer to browse randomly, we've got you covered.

## Features
- **Search by keyword**: Type in an ingredient or meal name (like "Chicken" or "Pasta").
- **Browse by country**: Explore dishes from around the globe using the dropdown filter.
- **Go random**: Feeling adventurous? Let the app pick a random recipe for you!
- **Vegetarian filter**: Easily hide dishes that contain meat with a simple toggle.
- **Save your favorites**: You can save any recipe to your Favorites list so you can easily find it later. It stays saved even if you refresh the page!
- **Detailed instructions**: Expand any recipe card to view step-by-step cooking instructions.

## Tech Stack
We built this project keeping things lightweight, fast, and simple without relying on heavy frontend frameworks!
- **HTML5**: For the basic structure of the layout.
- **Vanilla JavaScript**: All the logic, searching, routing, and saving favorites is powered by standard, pure JavaScript.
- **Vanilla CSS**: We wrote custom, pure CSS from scratch to style our application with a beautiful dark mode aesthetic (no external UI libraries used).
- **TheMealDB API**: All of the delicious recipe data is fetched live from this free, public API.

## How to run locally
1. Download or clone this repository.
2. Open the `index.html` file directly in your web browser. That's it!
