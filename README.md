# Recipe Finder

## Project Purpose
This is a web application built for my Web Application Programming course. The goal of the project is to build a functional UI that fetches data from a public API and uses JavaScript array higher-order functions to manipulate and display that data. 

## API Used
I am using [TheMealDB API](https://www.themealdb.com/api.php) to fetch recipe data, including meal names, categories, and images.

## Planned Features
* **Search:** A search bar so users can find specific recipes by name.
* **Filter:** Options to filter the loaded recipes by category (e.g., Vegetarian or Chicken).
* **Sort:** A feature to sort the displayed recipes alphabetically.
* These features will be implemented using JavaScript HOFs like `.filter()`, `.map()`, and `.sort()`.
